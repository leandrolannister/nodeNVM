#Gerenciador de versões nodeJs

Listar versões instaladas => nvm ls

Listar versões disponíveis para instalação => $ nvm ls-remote

Instalar uma versão => nvm install 8.15.0

Usar uma versão Instalada => nvm use 8.15.0


Desinstalar uma versão => nvm uninstall vX.X.X

Definir nome para uma versão => nvm alias meunome vX.X.X

Remover um nome de versão => nvm unalias meunome

Definir uma versão padrão => nvm alias default vX.X.X
